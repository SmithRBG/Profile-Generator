# Profile-Generator
Will generate profiles from input. (In progress)

## Usage
```
This application will output HTML with a list of profiles inputted from the command line prompts. 
You have the options of Manager, Engineer, or Intern.
```

## Dependencies
```
* node
* fs
* npm
```
## Languages
```
Javascript
html
css
```
